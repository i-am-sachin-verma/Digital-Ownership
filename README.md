# Digital Ownership

A sovereign identity platform powered by smart contracts. This project manages digital identities and access control using a decentralized architecture.

## Prerequisites
- Node.js (v18 or higher)
- [Foundry](https://getfoundry.sh/) (forge, cast, anvil)

## Smart Contracts Setup

Our smart contracts (`CoreLogic.sol`, `AccessController.sol`) are built and tested using Foundry.

1. **Install dependencies:**
```bash
   forge install
```
